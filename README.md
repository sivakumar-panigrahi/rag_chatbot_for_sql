# RAG SQL Chatbot

AI-powered chatbot that converts natural language into SQL queries, validates them, executes them on PostgreSQL, and explains the results in plain English.

## Tech Stack

- FastAPI
- Streamlit
- PostgreSQL
- Groq
- Hugging Face
- SQLAlchemy
- Alembic

## Features

- Natural Language to SQL
- SQL Validation & Safety
- Query Execution
- Result Explanation
- Analytics Dashboard
- Charts & Insights
- Conversation Persistence
